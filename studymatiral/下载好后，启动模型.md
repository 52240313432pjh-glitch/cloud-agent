对，那说明模型实际下载在这个路径：

```bash
/home/pjh/.cache/modelscope/hub/models/Qwen/Qwen2.5-7B-Instruct
```

你启动 vLLM 时就用这个本地路径，不要再写 `Qwen/Qwen2.5-7B-Instruct`，否则它可能又去 HuggingFace 或 ModelScope 查。

先这样启动：

```bash
export VLLM_USE_FLASHINFER_SAMPLER=0
export VLLM_USE_V1=0

vllm serve /home/pjh/.cache/modelscope/hub/models/Qwen/Qwen2.5-7B-Instruct \
  --served-model-name Qwen/Qwen2.5-7B-Instruct \
  --host 0.0.0.0 \
  --port 8001 \
  --dtype auto \
  --max-model-len 4096 \
  --gpu-memory-utilization 0.85
```

如果你显存是 16GB，7B 可能还是紧张。可以先确认显存：

```bash
nvidia-smi
```

如果报显存不够，就换 3B。下载 3B：

```bash
modelscope download \
  --model Qwen/Qwen2.5-3B-Instruct \
  --local_dir /home/pjh/.cache/modelscope/hub/models/Qwen/Qwen2.5-3B-Instruct
```

启动 3B：

```bash
export VLLM_USE_FLASHINFER_SAMPLER=0
export VLLM_USE_V1=0

vllm serve /home/pjh/.cache/modelscope/hub/models/Qwen/Qwen2.5-3B-Instruct \
  --served-model-name Qwen/Qwen2.5-3B-Instruct \
  --host 0.0.0.0 \
  --port 8001 \
  --dtype auto \
  --max-model-len 4096 \
  --gpu-memory-utilization 0.85
```

启动成功后测试：

```bash
curl http://localhost:8001/v1/models
```
